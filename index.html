# CAGR example
def cagr(begin, end, years):
    return (end / begin) ** (1 / years) - 1

print(f"CAGR (3y): {cagr(850e6, 1.2e9, 3):.2%}")

Financial Formula

\[\text{PV} = \sum_{t=1}^{N} \dfrac{FCF_t}{(1+r)^t} + \dfrac{TV}{(1+r)^N}\]

where \(TV = \dfrac{FCF_{N+1}}{r-g}\)
